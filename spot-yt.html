Spotify to Youtube Webapp

Timeline: 2025
Focus: UX/UI, Web development
Tools: Figma, Google cloud APIs, Spotify APIs, Microsoft visual studio Code; Typescript, SQLite, Redis, React JS, and Next JS

This web-converter tool utilizes Spotify and Youtube API keys in order to convert playlists from spotify to Youtube adding MTV feel for your Spotify playlists. By providing a URL to a public Spotify playlist you can transfer the playlist to your Youtube account which will be saved under your Youtube playlists. The tool handles caching using a database so that track IDs and artists are ported over faster the more the webapp is used. More details about the app and setup are available over on my GitHub
